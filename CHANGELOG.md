# Change Log

All notable changes to the "Roast My Code" extension will be documented in this file.

## [0.0.1] - 2026-02-03

### Added
- Initial release of Roast My Code
- Chat participant `@roast` for Copilot chat integration
- 11 roast levels from "Whisper" (1) to "These Go to Eleven" (11)
- **Workspace roasting** - roast your entire project when no file is open
  - Analyzes project structure, dependencies, configuration, and documentation
  - Use `@roast /workspace` to explicitly roast the workspace
- Special roast styles:
  - `/workspace` - Roast the entire workspace
  - `/gordon` - Gordon Ramsay kitchen nightmares style
  - `/shakespeare` - Shakespearean dramatic monologue
  - `/haiku` - Roasts delivered as haikus
  - `/explain` - Serious mode for actual fixes
- Context menu integration ("ðŸ”¥ Roast This Code")
- Status bar roast level indicator with quick picker
- Follow-up suggestions after roasts (including workspace option)
- Support for `#selection` and `#file` chat variables

### Philosophy
- Roast the code, not the coder
- Workplace-appropriate humor at all levels
- Educational AND entertaining feedback
